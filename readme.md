# Python program examples
These programs are a Backup of programs taught in my college. This repository exists as a way to backup and share these programs.

## Avalaible Programs:
### Practicals:
* expt1.py: Employee salary program. Calculates the net salary using employee's basic salary
* expt2.py: List methods. Example of the basic python list and its most common methods.
* expt3.py: Grading program, takes Marks as input. Has checking for invalid marks.

### Misc:
* inputs.py: Some examples of input function
* fibonacci.py: Generate fibonacci numbers upto n
* studentInfo.py: Read students info and marks. Prints the same with total and average marks
* salaryCompute.py: same as Expt 1
* evenNumbers.py: Read a range from user and print even numbers from it.

### Patterns
* rightAngleTriangle.py: Basic triangle using asterisks
* reverseTriangle.py: Basic upside down asterisk triangle 
* abcCube.py: Cube created using columns of charagters a, b and c
* charCube.py: Cube made using alphabets 'a' to 'i'
